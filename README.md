# 🎖️ EASY COMPANY | CODM Hub

> **Estado del Proyecto:** Operativo / Fase de Desarrollo v1.0
> **Objetivo:** Centralizar la información de despliegue, estrenos y coordinación de salas tácticas para Call of Duty: Mobile.

## 🛠️ Características del Sistema
* **Dashboard de Estrenos:** Actualizaciones en tiempo real sobre el nuevo contenido del juego.
* **Coordinación de Salas:** Enlace directo vía WhatsApp para la gestión de salas personalizadas y competitivas.
* **Panel Administrativo:** Sistema de gestión de contenidos (CMS) integrado para actualizaciones dinámicas de texto, multimedia y enlaces.
* **Interfaz Táctica:** Diseño "Dark Military" optimizado para una experiencia gaming inmersiva.
